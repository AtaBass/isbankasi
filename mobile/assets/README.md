Expo için aşağıdaki dosyaları ekleyin (veya `npx create-expo-app@latest dummy --template tabs` çalıştırıp dummy/assets içeriğini buraya kopyalayın):

- icon.png (1024x1024) — uygulama ikonu
- splash.png (1284x2778) — açılış ekranı (lacivert arka plan önerilir)
- adaptive-icon.png (1024x1024) — Android uyarlanabilir ikon

İsterseniz app.json içindeki "icon", "splash", "adaptiveIcon" alanlarını kaldırarak Expo varsayılanlarını kullanabilirsiniz.
